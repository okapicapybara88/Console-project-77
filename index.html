How to set up Visual Studio:

1.Go to this webpage to download Visual StudioVisual Studio download page

2.Download Visual Studio.

3.Open the .exe file.

4.Start the installation.

5.Let the installation complete.

6.Choose the software version.

7.Select the desktop version.

8.Wait for the files to be downloaded.

9.Reboot your PC.

And there you go, Visual Studio on your computer.
